# Dungeon Crawler — Parcial Final

## Integrantes
- Tomás Esteban Espinel Cohen
- Alejandro Mendez Santana

---

## Innovaciones
- Cinemáticas animadas en ASCII
- Audio multiplataforma
- Sistema de eventos aleatorios
- Sistema de puzzle con objetos e interacciones
- Inventario con combinación de objetos
- Puerta con múltiples objetos y código numérico
- Sistema de combate con minijuegos — Fantasmas
- Boss con 3 fases y minijuegos
- Imágenes 3D de objetos y enemigos
- Dos finales distintos
- Indicador visual de vida del jugador

---

## Compilación

### Windows
```bash
cd build
cmake .. -G "MinGW Makefiles"
mingw32-make
```

### Linux
```bash
cd build
cmake ..
make
```

### Mac
```bash
cd build
cmake ..
make
```
